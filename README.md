# v73 T100 A분할 월별 증액복리 패치

## 수정 목적
v72 A분할 백테스트는 차수 진입은 순차로 고쳤지만, 매월 1차/추가매수 금액이 고정되어 총자산이 커져도 투입금이 늘지 않는 문제가 있었습니다.

v73은 매월 리밸런싱 시점의 총자산을 기준으로 1차금액과 추가매수금액을 자동 증액합니다.

## 핵심 규칙
- 기준 초기자금: 기본 1.2억
- 기준 1차: 종목당 3,000만원
- 기준 추가: 종목당 -5%마다 500만원, 최대 6회
- 월별 증액복리 ON:
  - 월시작 총자산 / 초기자금 = 증액배율
  - 해당 월 1차금액 = 3,000만원 × 증액배율
  - 해당 월 추가금액 = 500만원 × 증액배율
- 차수는 v72 규칙 유지:
  - 직전 활성차수 매수가 대비 -5%마다 순차 진입
  - 한 종목당 하루 최대 1개 차수만 추가
- 익절:
  - 각 차수 +5% 익절
  - 1차는 익절 후 즉시 재진입 옵션 유지

## 예시
초기자금 1.2억으로 시작해 총자산이 1.44억이 되면 증액배율은 1.2배입니다.

- 종목당 1차: 3,000만원 → 3,600만원
- 종목당 추가: 500만원 → 600만원
- 최대 필요자금도 1.2배로 확대됩니다.

## 실행 방법
```text
streamlit run app.py
→ 10. T100 A분할 백테스트
→ 월별 증액복리 적용 체크
→ T100 A분할 v73 백테스트 실행
```

- 방공호/Turbo만 빠른 실행
- T100 기존형/확장형 백테스트
