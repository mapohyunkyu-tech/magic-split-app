# v71_RESTORE_2010_FAST_PLUS_T100_A_TP5

## 수정 목적
v70에서 `10. T100 A분할 백테스트`를 붙이면서 v68에 있던 장기검증/빠른실행 기능이 빠진 문제를 복구한 버전입니다.

## 복구/유지된 기능
- `6. 섹터전략 백테스트`의 `2010부터 장기검증` 복구
- `최대 검증 거래일 5000` 계열 장기검증 유지
- `방공호/Turbo만 빠른 실행` 버튼 복구
- `T10/T100 Turbo 공격ETF 100% NO CTA` 유지
- `T10/T100 Turbo 확장형 SP500+BOND` 유지
- `7-1. T100 하이브리드 운용모드` 유지
- `10. T100 A분할 백테스트` 유지

## T100 A분할 v70 규칙
- 월초 상위 2개 ETF 선택
- 1종목당 3,000만원 1차 진입
- 각 종목이 월초 진입가 대비 -5%마다 500만원 추가
- 종목별 최대 6회 추가
- 각 차수 +5% 익절
- 1차는 +5% 익절 후 즉시 재진입 가능
- 월말/월초 리밸런싱 시 남은 차수 정리 후 새 월 상위 2개로 교체

## 백테스트 실행
### 기존 T100 / 확장형 T100
`6. 섹터전략 백테스트` → `2010부터 장기검증` → 방공호 방식 선택 → `방공호/Turbo만 빠른 실행`

### A분할 백테스트
`10. T100 A분할 백테스트` → 설정 확인 → `T100 A분할 v70 백테스트 실행`

## 주의
T100/Turbo만 빠르게 보려면 반드시 `방공호/Turbo만 빠른 실행`을 사용하세요. 일반 섹터전략 백테스트 실행은 개별주 섹터전략까지 계산해서 느릴 수 있습니다.
