## 핵심 추가

- 기존 `섹터신뢰도 자동가중` 버전에 `강한차단형`과 `초강한차단형`을 추가했습니다.
- 목적은 좋은 섹터에 더 싣는 것뿐 아니라, 최근 손실이 누적되고 현재 유입이 약한 섹터를 더 강하게 줄이는 것입니다.
- 섹터 영구 제외가 아니라, 거래대금·쏠림·현재대장 흐름이 살아나면 다시 B등급으로 복귀하도록 설계했습니다.

## 섹터 자동가중 모드

- OFF: 기존과 동일
- 보수형: A 1.0배 / B 1.0배 / C 0.5배 / D 0배
- 표준형: A 1.2배 / B 1.0배 / C 0.5배 / D 0배
- 공격형: A 1.3배 / B 1.0배 / C 0.6배 / D 0배
- 강한차단형: A 1.2배 / B 1.0배 / C 0.3배 / D 0배
- 초강한차단형: A 1.2배 / B 1.0배 / C 0배 / D 0배

## 강한차단 조건

강한차단형/초강한차단형에서는 아래 조건을 더 강하게 반영합니다.

- 최근 60일 또는 120일 섹터 실현손익이 마이너스
- 최근 60일 손실 매도 비율이 높음
- 전량회수/추가매수금지/신규금지 등 유입 약화 신호
- 현재 거래대금·쏠림 회복이 약함

단, 현재 유입이 다시 살아나면 C/D 섹터도 B등급으로 복귀합니다.
