## 이번 보정 핵심

이전 백테스터에서 날짜별로 전체 대표주 계산을 반복하면서 `계산실패`가 많이 뜨던 구조를 바꿨습니다.

- 종목별 가격 데이터를 시작 전에 1회만 로딩
- 기본 백테스트는 섹터별 핵심 대표주 3개로 압축
  - 대장주 1개
  - 2등대표주 1개
  - 회전형 1개
- 날짜별 신호는 로딩된 가격맵으로 계산
- 기존 `계산실패` 반복을 줄이기 위해 빠른 전용 신호 계산 함수를 사용
- 익절/발빼기/대장주 재진입대기/재진입허용 흐름 추가

## 유지한 것

- 기존 `2. 운영판단기`
- 기존 `3. TOP50`

- 기존 대표주유니버스 CSV

## 추가/보정한 것

### 6. 섹터전략 백테스트

섹터 순환매 판단기의 `오늘행동` 신호를 과거 날짜별로 계산하고, 다음 거래일 시가로 매매한 것으로 검증합니다.

백테스트 기본 원칙:

- 신호일 장마감 기준으로 섹터 순환매/대표주 흐름 계산
- 실제 매수/매도는 다음 거래일 시가 기준
- 초보 기본값은 동시 1섹터 진입
- 방어예수금 아래로 내려가는 매수는 차단 또는 수량 축소
- 대장주, 2등대표주, 회전형을 분리해서 차수 관리

## 기본 차수 규칙

### 대장주

- 1차: 500만
- 2차: 400만
- 3차: 300만
- 2차 기준: 1차 진입가 대비 -5%
- 3차 기준: 1차 진입가 대비 -10%
- 최대 3차까지만 허용

### 2등대표주

- 1차: 300만
- 2차: 300만
- 2차 기준: 1차 진입가 대비 -5%
- 최대 2차까지만 허용

### 회전형

- 1차: 200만
- 2차 없음
- 물타기 없음

## 익절 규칙

- 대장주: +8% 절반 익절, +12% 잔여 정리
- 2등대표주: +5% 익절
- 회전형: +3% 익절

## 발빼기/재진입 규칙

- `일부팔기`: 대장주는 절반만 줄이고 `재진입대기` 상태로 둠
- `일부팔기`: 2등대표주/회전형은 전량 회수
- `전량회수`: 해당 종목/섹터 전량 정리
- 대장주 재진입: 발빼기 후 섹터가 다시 `사기`로 회복되고 현재역할이 `현재대장`이면 1차만 복원
- 같은 날 판 종목은 같은 날 다시 사지 않음
