# v27_SECTOR_STRATEGY_BACKTEST_20260629

매직스플릿 v27 계열에 `6. 섹터전략 백테스트` 메뉴를 추가한 버전입니다.

## 유지한 것

- 기존 `1. 요양원`
- 기존 `2. 운영판단기`
- 기존 `3. TOP50`
- 기존 `4. 보유종목 판단기`
- 기존 `5. 섹터 순환매 판단기`
- 기존 대표주유니버스 CSV

## 추가한 것

### 6. 섹터전략 백테스트

섹터 순환매 판단기의 `오늘행동` 신호를 과거 날짜별로 계산하고, 다음 거래일 시가로 매매한 것으로 검증합니다.

백테스트 기본 원칙:

- 신호일 장마감 기준으로 섹터 순환매/대표주 흐름 계산
- 실제 매수/매도는 다음 거래일 시가 기준
- 초보 기본값은 동시 1섹터 진입
- 방어예수금 아래로 내려가는 매수는 차단 또는 수량 축소
- 대장주, 2등대표주, 회전형을 분리해서 차수 관리

## 기본 차수 규칙

### 대장주

- 1차: 500만
- 2차: 400만
- 3차: 300만
- 2차 기준: 1차 진입가 대비 -5%
- 3차 기준: 1차 진입가 대비 -10%
- 최대 3차까지만 허용

### 2등대표주

- 1차: 300만
- 2차: 300만
- 2차 기준: 1차 진입가 대비 -5%
- 최대 2차까지만 허용

### 회전형

- 1차: 200만
- 2차 없음
- 물타기 없음

## 매도 규칙

- `일부팔기`: 대장주는 절반, 2등대표주/회전형은 전량 회수
- `전량회수`: 해당 종목/섹터 전량 정리
- `신규금지`: 새 매수 금지
- `추가매수금지`: 기존 보유분 물타기 금지

## 결과 탭

- 요약
- 자금흐름
- 매매내역
- 최종보유
- 실패구간

## 주의

백테스트는 FinanceDataReader 일봉 데이터를 사용합니다. 무료 서버에서는 오래 걸릴 수 있으므로 처음에는 20~40거래일로 빠른 검증을 권장합니다.
