# MAGIC V27 A GRADE STRONG RECOVERY FIX

`v27_TOP50_UNIVERSE_GRADE_FIX_20260624` 기반 보완판입니다.

## 수정 이유

대덕전자처럼 보유차수 판단기에서 자동등급 A이고, 추가매수 기준가도 이미 도달했으며, 공포매수 슬롯도 열려 있는데도 `강한 회수모드` 차단 로직이 먼저 적용되어 `⛔ 금지`로 표시되는 문제가 있었습니다.

예: 대덕전자
- 자동등급 A
- 기준도달여부 도달
- 추가매수가능개수 1개
- 기존 결과: `⛔ 금지 / 강한 회수모드: 공포매수보다 현금방어 우선`

## 수정 내용

- 강한 회수모드에서도 본전권/약수익 종목은 기존처럼 회수후보 유지
- 손실 구간 종목은 무조건 금지로 끝내지 않고 차수 기준/자동등급 판단까지 내려보냄
- `A급 + 기준가 도달 + 공포매수 슬롯 있음`이면 `🟢 오늘매수가능`으로 표시 가능
- B급은 기존처럼 `🟠 B급 도달검토` 유지
- 신규 TOP50 매수와 금액확장은 계속 금지/잠금
