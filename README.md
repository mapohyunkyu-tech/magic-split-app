# MAGIC V27 PLAIN KOREAN FEAR BUY GUIDE

`v27_OPERATION_FEAR_BUY_WORDING_FIX_20260619` 기반 보완판입니다.

## 수정 이유

운영판단기와 보유차수 판단기의 문구가 서로 충돌해 보였습니다.
사용자가 보기에는 운영판단기는 “공포매수 하지 마”라고 하고,
보유차수 판단기는 “공포매수 하라”고 하는 것처럼 보여 혼란이 있었습니다.

## 핵심 정리

앱의 실제 의미는 아래와 같습니다.

- TOP50 신규매수: 금지 또는 축소
- 새 종목 확장: 금지
- 15만원 → 20만원/25만원 금액확장: 잠금
- 기존 보유종목 공포매수: 조건부 허용
- 조건: A급 + 마지막 차수 기준가 도달 + 공포매수 탄창 한도
- B급 도달검토: 바로 매수 신호가 아니라 수동검토
- 기준가대기: 아직 기다림

## 추가 보완

- 강한 회수모드에서도 예수금이 절대방어선 위에 있으면 A급 기준도달 1개까지만 예외 검토 가능하게 문구/로직을 정리했습니다.
- A/B급 기준도달이 0개면 “공포매수 비허용” 대신 “오늘은 살 놈이 없음”으로 안내합니다.
- 보유차수 판단기 상단에 사람 말로 풀어쓴 오늘 행동 요약을 추가했습니다.
