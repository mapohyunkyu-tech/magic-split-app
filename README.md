# v58 T100 HYBRID 1↔3 LIVE MANUAL MODE FIX

## 수정 이유
운용모드에서 `현재 운용모드 = 1순위 공격모드`를 선택했는데도, 현재 총자산이 직전 잠금 기준금액 대비 +50%를 넘으면 자동으로 `3순위 6310 신규 전환`으로 판정되는 문제가 있었습니다.

예: 직전 잠금 기준금액 10,000,000원, 현재 총자산 16,000,000원 입력 시 앱은 +60% 수익으로 인식하여 6310 가능권으로 판단합니다.

## v58 수정 내용
- `자동판정` 모드일 때만 +50% 수익잠금 조건으로 6310 자동 전환합니다.
- `1순위 공격모드`를 직접 선택하면 +50% 조건이 충족돼도 1순위 공격모드를 유지합니다.
- `3순위 6310 잠금모드`를 직접 선택한 경우에는 +50% 조건 충족 또는 이미 6310 진입 체크가 있을 때만 6310을 허용합니다.
- 외부 추가입금이 수익으로 오해되지 않도록 안내문을 추가했습니다.

## 입력 주의
현재 총자산에는 T100 실험계좌 안의 돈만 넣으세요. 외부에서 새로 넣은 현금을 함께 입력하면 앱은 그것을 수익으로 인식합니다.

예: 처음 1,000만원으로 시작했고 외부에서 600만원을 추가 입금했다면,
- 현재 총자산: 16,000,000
- 직전 잠금 기준금액: 16,000,000
으로 맞춰야 6310이 잘못 켜지지 않습니다.

진짜 수익으로 1,000만원이 1,600만원이 된 경우라면,
- 현재 총자산: 16,000,000
- 직전 잠금 기준금액: 10,000,000
으로 입력하면 6310 가능권이 맞습니다.
