  - 강한상승장 1.7배

  - 조정장 0.5배, 대장주만

### 2) MDD -10% 수익확대 프리셋
- 직접입력/현재값
- A 상승배율강화
- B 대장주1차확대
  - 대장주 1차 1,300만 / 2차 500만 / 3차 300만
- C 조합공격
  - 대장주 1차 1,300만 / 2차 500만 / 3차 300만
  - 2등대표주 1차 700만 / 2차 300만
  - 대장주 +10% / +18% 익절
- D 대장주25%추세익절
  - 대장주 +10% 30% 익절
  - 대장주 +18% 30% 익절
  - 대장주 +25% 잔여 전량익절

### 3) 대장주 익절방식 추가
- 2단계: 기존 방식, TP1 일부익절 + TP2 전량익절
- 3단계추세익절: TP1 일부익절 + TP2 일부익절 + TP3 잔여익절

### 4) settings CSV 표시 보정
- 장세필터 프리셋의 실제 적용 배율이 settings CSV에 정확히 찍히도록 수정했습니다.
