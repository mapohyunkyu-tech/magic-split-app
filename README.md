# Magic Split v51 - 강환국식 MDD5 이동평균 자산배분 백테스트 추가

## 추가 목적
한국인이 만든 전략 후보 중 `연복리 15% / MDD 5% 추구`로 알려진 강환국식 이동평균 자산배분을 매직스플릿 앱 안에서 검증하기 위한 모드입니다.

## 추가 위치
`6. 섹터전략 백테스트` 메뉴 안에 아래 섹션이 추가됩니다.

- `1-0-3) 강환국식 MDD5 이동평균 자산배분 백테스트`

## 테스트 프로필

### 1) 강환국 MDD5 US/KR 프록시
원형에 가깝게 아래 5개 슬롯을 사용합니다.

- IWM: 미국 소형주
- KOSDAQ150: 한국 소형주/성장주 대체
- TLT: 미국 장기채
- 한국채권: 국내 국고채/단기채 ETF 프록시
- GLD: 금

### 2) 강환국 MDD5 한국 ETF 프록시
국내 상장 ETF로 구현하기 위한 대체형입니다.

- KODEX200 / KOSPI200
- KOSDAQ150
- 국내상장 NASDAQ100
- GOLD
- DOLLAR
- BOND

## 운용 규칙
- 월말 종가 기준으로 각 자산의 N개월 이동평균을 계산합니다.
- 해당 자산 가격이 N개월 이동평균 위에 있으면 해당 슬롯을 보유합니다.
- 이동평균 아래에 있으면 해당 슬롯을 CASH로 대체합니다.
- 월말 신호는 다음 거래일부터 반영합니다.
- 상장 전 데이터가 없거나 lookback 부족이면 해당 슬롯은 CASH 처리됩니다.

## 비교 옵션
- 3개월 이동평균
- 6개월 이동평균
- 10개월 이동평균

기본값은 3/6/10개월을 모두 한 번에 비교합니다.

## 데이터 모드
US/KR 혼합 프록시의 미국 ETF는 기본적으로 Yahoo Adj Close 전용 정확검증 모드를 사용합니다.
필요하면 `Yahoo 우선 + FDR/Stooq 백업` 모드로도 돌릴 수 있습니다.

## 출력 파일
실행 후 ZIP 안에 다음 파일이 생성됩니다.

- `magic_split_KANG_MDD5_MA_summary_YYYY-MM-DD.csv`
- `magic_split_KANG_MDD5_MA_data_status_YYYY-MM-DD.csv`
- 전략별 daily CSV
- 전략별 signals CSV
- 전략별 rebalances CSV

## 실행 방법
```bash
streamlit run app.py
```

앱에서:

4. US/KR 혼합 프록시 / 한국 ETF 프록시 체크
5. 3/6/10개월 선택
6. `강환국 MDD5 MA 빠른 백테스트 실행`
