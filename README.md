# v62 T100 HYBRID LIVE - CAP 신호 자동계산 수정

## 수정 내용

- `CAP5 또는 5일 누적 -6% 방어 신호` 체크박스를 제거했습니다.
- 이제 사용자가 CAP 신호를 직접 체크하지 않습니다.
- 앱이 아래 값으로 자동 계산합니다.
  - 현재 운용기준금액 = 현재 T100 평가금액 + 전략 CASH/방어현금
  - 전일 운용기준금액
  - 5거래일 전 운용기준금액
- 하루 변동률이 -5% 이하이거나 5거래일 변동률이 -6% 이하이면 자동 방어 ON입니다.
- 계좌 예수금 메모는 계속 계산에서 제외합니다.
- Streamlit DeltaGenerator/help 표가 뜨던 삼항 표시 문제도 방지했습니다.

## 천만원 실험 기본 입력

- 실험/운용 원금 메모: 10,000,000
- 현재 T100 평가금액: 실제 T100 평가금액
- 전략 CASH/방어현금: 1순위 공격 중이면 0
- 계좌 예수금 메모: 전략 외 예수금만 입력
- 직전 잠금 기준금액: 10,000,000
- 현재 운용모드: 1순위 공격모드
- 전일/5거래일 전 운용기준금액: 모르면 현재 운용기준금액과 같게 입력

## 실행

```bash
streamlit run app.py
```
