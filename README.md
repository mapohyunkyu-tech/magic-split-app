# v27_SECTOR_LIVE_OPERATION_BOARD_HOLDING_LEDGER_20260629

## 핵심 변경

- `8. 실전 보유장부` 메뉴 추가
- 실제 매수한 종목의 `차수 / 매수가 / 매수금액 / 보유주수` 기록
- 기존 보유장부 CSV 불러오기/저장
- 실전 운영판의 `magic_split_live_stock_action_날짜.csv` 업로드 시 보유 종목만 신호 매칭
- 보유 종목에 대해 `팔아라 / 줄여라 / 대기`를 백테스트 룰대로 표시
- `팔아라`는 전량매도, `줄여라`는 50% 줄이기 기본 처리
- 매도/축소 기록 시 보유주수 자동 차감 및 실현손익 기록
- 보유장부/보유요약/매도기록 CSV 묶음 다운로드

## 사용 순서

1. `7. 실전 운영판`에서 오늘 신호 계산
2. 산 종목은 `8. 실전 보유장부`에 입력
   - 종목, 코드, 섹터, 현재역할, 차수, 매수가, 매수금액, 주수
3. 다음날부터 `7. 실전 운영판`에서 받은 `stock_action CSV`를 `8. 실전 보유장부`에 업로드
4. 보유 중인 종목만 `팔아라 / 줄여라 / 대기` 확인
5. `팔아라` 또는 `줄여라` 실행 후 매도/축소 기록
6. 장부 CSV 묶음 다운로드해서 다음날 다시 불러오기

## 백테스트 룰 반영

- 현재대장 + 사기 = 사라
- 현재2등대표 + 사기 = 사라
- 현재회전형 + 사기 = 회전형 OFF이므로 사지마
- 일부팔기 = 줄여라
- 전량회수 = 팔아라
- 추가매수금지 = 추가매수 하지마
- 신규금지 = 사지마
