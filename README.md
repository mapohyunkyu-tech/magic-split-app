# MAGIC V27 SIMPLE HOLDING OVERRIDE FIX

`v27_FEAR_BUY_EXCEPTION_FIX_20260619` 위에 간편보유 적용 오류를 보정한 버전입니다.

## 수정 이유

기존 버전은 정밀 보유차수에 이미 있는 종목이면 `보유간편` 입력을 판단에서 제외했습니다.
그래서 사용자가 새로 산 차수를 `서진시스템,2,69200`처럼 간편 입력해도, 기존 1차 정밀 데이터가 있으면 새 차수가 판단표에 반영되지 않았습니다.

## 수정 내용

- `보유간편`이 정밀 보유차수에 있는 종목도 보정 가능
- 간편 현재차수가 정밀 최고차수와 같거나 높으면 판단표 최고차수를 간편값으로 교체
- 정밀 보유차수 데이터는 삭제하지 않고 그대로 유지
- 간편으로 새 차수를 올린 경우 마지막진입가가 있으면 수량 1주 기준으로 전체 평균단가 참고값에도 반영
- 간편 저장/수정/삭제 후 `st.rerun()`으로 화면 카운트 즉시 갱신
- 진단에 `간편판단반영`, `간편정밀차수보정`, `간편신규판단`, `간편무시_정밀보다낮은차수` 표시

## 입력 예시

```text
서진시스템,2,69200
삼성E&A,3,51600
동진쎄미켐,3
```

## 주의

간편보유는 공포매수/다음 차수 기준가 판단용입니다.
정확한 총수량, 평균단가, 손익률은 나중에 정밀 보유차수로 입력하면 더 정확해집니다.
