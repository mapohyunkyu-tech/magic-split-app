# v61 T100 HYBRID 1↔3 운용모드 - 70/30 자동계산 수정

## 핵심 수정

`전략 CASH/방어현금`을 사용자가 직접 70/30으로 계산하지 않아도 되도록 수정했습니다.

이제 앱은 아래 금액만 기준으로 목표금액을 자동 계산합니다.

```text
운용 계산 기준 = 현재 T100 평가금액 + 전략 CASH/방어현금
```

`계좌 예수금 메모`는 계산에서 제외됩니다.

## 예시

### 1,150만 원에서 방어모드 70/30

```text
현재 T100 평가금액: 11,500,000
전략 CASH/방어현금: 0
계좌 예수금 메모: 6,000,000  # 계산 제외
현재 운용모드: 1순위 방어모드
```

앱 목표값:

```text
T100 목표: 8,050,000
전략 CASH 목표: 3,450,000
T100 매도 필요: 3,450,000
```

### 1순위 공격모드

```text
T100 목표: 100%
전략 CASH 목표: 0%
```

### 3순위 6310 잠금모드

```text
T100 목표: 60%
전략 CASH 목표: 40%
```

## 메뉴 위치

```text
7-1. T100 하이브리드 운용모드
```

## 실행

```bash
streamlit run app.py
```

## 주의

- 계좌에 남아 있는 별도 예수금은 `계좌 예수금 메모`에만 입력하세요.
- 방어모드 때는 `전략 CASH/방어현금`을 직접 계산하지 말고, 앱의 리밸런싱 표에서 목표금액과 매도/매수 금액을 확인하세요.
