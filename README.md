# Magic Split v27 - 0/7/3 방공호형 공격복리 백테스트 추가

## 버전
v27_SECTOR_LIVE_OPERATION_BOARD_BUNKER_073_FINAL_FIX_20260701

## 추가된 핵심 기능
`1-0) 0/7/3 · 5/3/2 · 70/20/10 방공호+부스터 통합 백테스트`에 `0/7/3 방공호형 공격복리` 모드를 추가했습니다.

## 073 구조
초기 1억 기준:

- 대장주엔진 0% = 0원
- 진짜방공호 70% = 7,000만
- 수익부스터 30% = 3,000만

## 운용 방식
월간 상위 70:30 강제 리밸런싱은 하지 않습니다.
각 칸은 자기 안에서 복리로 굴러갑니다.

- 방공호 수익 → 방공호 내부 복리
- 부스터 수익 → 부스터 내부 복리
- 대장주 엔진 → 사용하지 않음

## 분기 밴드 점검
분기 첫 거래일에만 상위 비중을 점검합니다.

허용 밴드:

- 진짜방공호 65~80%
- 수익부스터 20~35%

조정 규칙:

- 부스터가 35% 초과 → 초과분을 방공호로 잠금
- 방공호가 65% 미만 → 부스터에서 방공호를 70% 근처로 보충
- 부스터가 20% 밑으로 내려가도 하락장 칼날잡기를 피하기 위해 자동 보충하지 않음
