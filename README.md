# v56 T100 HYBRID 1↔3 LIVE MAGIC CLEAN

수정 내용:
- Streamlit API 도움말표(text_input, time_input, write 등)가 화면에 떠 있는 문제 방지.
- 패키지에 `.streamlit/config.toml` 추가: `[runner] magicEnabled = false`.
- app.py에도 `STREAMLIT_RUNNER_MAGIC_ENABLED=false` 안전장치 추가.
- T100 HYBRID 1↔3 운용모드 v55의 6310 +50% 게이트는 그대로 유지.

실행:
1. ZIP을 풀고, 반드시 ZIP을 푼 폴더 안에서 실행하세요.
2. `streamlit run app.py`
3. 이미 떠 있던 브라우저 창은 새로고침하지 말고 닫은 뒤 새 주소로 접속하는 것을 권장합니다.

운용 기준:
- 천만원 시작 기준 6310 전환 목표는 1,500만원입니다.
- 1,500만원 전에는 3순위 6310 잠금모드로 움직이지 않습니다.
- 그 전에는 1순위 T100 실전형만 운용합니다.
