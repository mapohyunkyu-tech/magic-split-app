# v55 T100 HYBRID 1↔3 LIVE OPERATION GATE FIX

## 수정 이유
- Streamlit 화면에 `text_input`, `time_input`, `write` 같은 Streamlit API 도움말표가 보이는 표시 버그 방지.
- T100 HYBRID 1↔3 운용모드에서 3순위 6310은 수익잠금 조건을 충족하기 전에는 불가능하도록 명확화.

## 핵심 규칙
- 1순위: T100 CAP5 + 5일 누적 -6% 방어
  - 평상시 T100 100%
  - 하루 -5% 또는 5일 누적 -6% 신호 시 T100 70% + CASH 30%
- 3순위: T100 SPLIT 60/30/10 + C안
  - 운용판에서는 T100 60% + CASH 40%
- 3순위 전환 조건
  - 직전 잠금 기준금액 대비 +50% 달성 때만 가능
  - 예: 기준 1,000만원이면 총자산 1,500만원, 즉 수익 +500만원부터 가능

## v55 변경사항
- `7-1. T100 하이브리드 운용모드 v55`로 표시
- 현재 총자산이 6310 목표금액 미만이면 `6310 불가` 경고 출력
- 실수로 3순위 잠금모드를 선택해도, `이미 6310 잠금에 진입한 계좌`를 체크하지 않고 수익조건도 미달이면 자동으로 1순위 쪽으로 돌림
- 결과표에 `6310전환목표금액`, `6310전환까지남은금액`, `6310가능여부` 추가
- Streamlit magic 표시 방지용 안전장치 추가
