# Magic Split v27 - 70/20/10 방공호+부스터 분리 백테스트

## 수정 핵심

기존 70/30 듀얼모멘텀은 방공호 30%가 NASDAQ100/KOSDAQ150 같은 주식형 자산을 많이 잡아 수익은 커졌지만 폭락장 방어 역할은 약했습니다.

이번 버전은 70/30을 다음처럼 분리합니다.

- 대장주 엔진 70%
- 진짜 방공호 20%
  - CASH 50% 기본 완충
  - GOLD/DOLLAR 방어형 듀얼 50%
- 수익 부스터 10%
  - KODEX200 / KOSDAQ150 / NASDAQ100 / GOLD / DOLLAR 공격형 듀얼 상위 2개

## 사용 방법

6. 섹터전략 백테스트에서:

- 70/30 방공호 같이 계산: ON
- 방공호 방식: 70/20/10 방공호+부스터
- 시장붕괴 차단 필터: OFF 권장(순수 효과 먼저 확인)
- 성장주 보조필터: OFF 권장
- 기간: 2020부터 전체검증 또는 빠른검증 60거래일
