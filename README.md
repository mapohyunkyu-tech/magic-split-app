# MAGIC V27 FEAR BUY ALLOWED / EXPANSION LOCK

`v27_FEAR_BUY_EXPANSION_GUARD_20260619` 기반 패치입니다.

## 핵심 수정

이번 버전은 사용자의 매직스플릿 철학을 더 정확히 반영합니다.

- 금액확장과 공포매수를 완전히 분리
- 예수금이 줄고 평가손실이 나는 구간에서는 **금액확장 잠금**
- 그러나 공포매수 자체는 막지 않음
- 기존 보유종목 중 앱이 자동 판단한 A/B급이 마지막 차수 기준가에 도달했을 때만 기본금액으로 확인
- `공포매수제한` 문구를 `공포매수허용` 중심으로 수정

## 운영 원칙

### 하락/손실 구간

- 신규매수: 금지 또는 축소
- 금액확장: 잠금
- 공포매수: 허용
- 조건: 기존 보유종목 + A/B급 + 마지막 차수 기준가 도달 + 예수금 방어선 유지

### 안정 구간

다음 조건을 충족할 때만 15만원 → 20만원/25만원 같은 금액확장을 검토합니다.

- 목표종목수 도달
- 장부자산 기준 충족
- 예수금 여유선 이상
- 예수금비중 충분
- 평가손익률 안정
- 운영모드 정상운용

## 표시 문구

- 확장상태: `확장잠금` 또는 `확장허용`
- 공포매수상태: `공포매수허용`, `일반운용`, `현금방어우선`, `공포매수보류`
- 오늘적용매수금액: 확장잠금이면 기존 기본매수금액 유지
