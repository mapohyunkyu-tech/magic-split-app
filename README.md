# MAGIC V27 B GRADE RECOVERY CONFIRM

`v27_A_GRADE_STRONG_RECOVERY_FIX_20260624` 기반 보완 패치입니다.

## 수정 이유

사용자 계좌가 대부분 B/C 등급일 때 A급 기준도달만 기다리면 반등장에서도 계속 매수 후보가 0개로 남는 문제가 있었습니다.

## 수정 내용

- `🟣 B급 회복확인` 칸 추가
- 조건: B급 + 손실권 + 당일 +2% 이상 반등 + 거래대금/회전 양호 + 과열/요양원 아님
- B급 회복확인은 자동매수 신호가 아니라 수동검토 후보입니다.
- A급 후보가 없거나 슬롯이 남고, 탄창 여유가 있을 때만 1개 검토하는 용도입니다.
- C급/요양원/과열주는 반등해도 회복확인 후보에서 제외합니다.

## 기존 기능 유지

- TOP50 700개 유니버스 보정
- A급 강한 회수모드 예외 공포매수
- B급 도달검토
- 기준가대기
- 간편보유/차수보정
- 공포매수 탄창제
