# MAGIC v27 SECTOR AUTO LEARNING MAP

v27 섹터신뢰/테마실체 리스크 필터에 **섹터미분류 자동수집 + 섹터매핑 우선반영**을 추가한 버전입니다.

## 핵심 변경

- TOP50/보유차수 공통 섹터엔진 유지
- `섹터매핑` Google Sheets 탭을 최우선 섹터 원천으로 사용
- TOP50/보유차수에서 `기타`, `낮음`, `수동확인필요`로 나온 종목을 `섹터미분류` 탭에 자동 누적
- `섹터미분류` 탭에 추천섹터/추천신뢰/추천사유 자동 생성
- `5. 섹터진단` 메뉴에서 추천섹터를 `섹터매핑` 탭에 자동 반영 가능
- 섹터신뢰 낮음 신규매수 제한 유지
- 아이로보틱스 같은 테마실체 확인필요 종목 방어 유지
- `최종판정=금지`인데 `오늘매수=매수가능`으로 남는 버그 방지 유지
- LG, 두산에너빌리티 등 obvious 대형주 코드확정맵 유지

## 새 Google Sheets 탭

### 섹터매핑

앱이 가장 먼저 읽는 최우선 섹터맵입니다.

```text
종목코드 | 종목명 | 그룹 | 섹터신뢰 | 메모
```

예:

```text
003550 | LG | 지주회사/그룹지주 | 확정 | 코드확정맵
034020 | 두산에너빌리티 | 원전/발전설비/에너지 | 확정 | 코드확정맵
028050 | 삼성E&A | 플랜트/EPC | 확정 | 수동보정
```

### 섹터미분류

TOP50/보유차수 실행 중 기타/낮음으로 나온 종목이 자동 누적됩니다.

```text
최초발생일 | 최근발생일 | 발생원천 | 발생횟수 | 종목코드 | 종목명 | 현재그룹 | 현재신뢰 | 현재판정 | 추천섹터 | 추천신뢰 | 추천사유 | 테마실체 | 테마리스크 | 점수 | 20일수익률 | 60일수익률 | 처리상태 | 메모
```

## 사용 흐름

1. TOP50 또는 보유차수 판단기를 실행합니다.
2. 기타/낮음 종목이 있으면 앱이 자동으로 `섹터미분류` 탭에 쌓습니다.
3. `5. 섹터진단` 메뉴에서 `섹터미분류 탭 보기`를 엽니다.
4. `추천섹터를 섹터매핑에 자동 반영` 버튼을 누르면 추천 가능한 종목만 `섹터매핑`에 들어갑니다.
5. 다음 TOP50/보유차수 실행부터 `섹터매핑`이 최우선 적용됩니다.

## 외부 원천 우선순위

1. Google Sheets `섹터매핑`
2. 내장 코드확정맵 / 종목명확정맵
3. KIND/FDR 업종·주요제품
4. 종목명/사업 키워드
5. 구버전 키워드 보조
6. 그래도 실패하면 `기타 / 낮음 / 수동확인필요`

## 뉴스/공시 선택 설정

외부 뉴스/공시 자동 점검을 TOP50 계산에 반영하려면 Streamlit secrets에 아래를 추가하세요.

```toml
THEME_CRAWL_ENABLED = "true"
NAVER_CLIENT_ID = "네이버_개발자센터_Client_ID"
NAVER_CLIENT_SECRET = "네이버_개발자센터_Client_Secret"
DART_API_KEY = "OpenDART_API_KEY"
```

키가 없거나 `THEME_CRAWL_ENABLED`가 없으면 앱은 기존 내장 섹터/테마 규칙만 사용하고 정상 작동합니다.

## 주의

- 네이버 종목토론실은 욕/찬티/안티 노이즈가 커서 자동 감점 원천으로 쓰지 않았습니다.
- 추천불가 또는 테마리스크 고위험 후보는 자동 반영하지 않고 `섹터미분류`에 대기시킵니다.
- 문법검사는 `python -m py_compile app.py`로 통과했습니다.
