# MAGIC V27 FEAR BUY AMMO SYSTEM

`v27_SIMPLE_HOLDING_OVERRIDE_FIX_20260619` 기반 패치입니다.

## 수정 이유

120종목 보유 상태에서 회수모드라고 해서 `하루 2개`로 고정하면 너무 단순합니다.
반대로 하락장에 5~10개씩 무작정 사면 예수금이 먼저 마릅니다.

그래서 이번 버전은 **금액확장/신규매수는 잠그고, 기존 보유종목 공포매수만 탄창제로 조절**합니다.

## 핵심 철학

- 신규매수: 제한 또는 금지
- 금액확장: 예수금 안정/평가손실 안정 전까지 잠금
- 공포매수: 기존 보유종목만 허용
- 매수대상: A급 + 마지막 차수 기준가 도달 우선
- 매수금액: 15만원 기본금액 유지
- 매수개수: 고정 2개가 아니라 예수금/후보개수/시장급락에 따라 동적 결정

## 공포매수 탄창제

앱이 아래 값을 계산합니다.

- 공포매수 방어예수금
- 공포매수 사용가능탄창
- 공포매수 최대슬롯
- A급 기준도달 개수
- 시장급락레벨
- 오늘 최종 공포매수 슬롯

대략적인 작동 방식:

- 예수금비중 10% 미만 또는 절대방어선 하회: 0개
- 예수금비중 15% 미만: 최대 1개
- 예수금비중 20% 미만: 최대 2개
- 손실 -10%대이고 현금 여유가 있으면: 후보/시장급락에 따라 2~4개
- A급 기준도달이 많고 시장 급락이면: 최대 4~5개까지 가능

## 화면 표시

보유차수 판단기에서 다음 정보를 추가 표시합니다.

- 공포매수 탄창: 방어예수금 / 사용가능탄창 / 최대슬롯
- 공포매수 탄창판정: 최종슬롯 / A급 기준도달 / B급 / 시장급락레벨

## 기존 기능 유지

- 보유간편 입력
- 간편 보유가 정밀 보유차수보다 같거나 높은 차수면 판단용 보정
- 정밀 보유차수는 삭제하지 않음
- 종목별 최고차수 1줄 표시
- 추가매수기준가/현재장도달시
- 금액확장 잠금
- 회수/익절검토 +5% 기준
