# Magic Split v27 - 70/30 방공호 통합 백테스트

## 추가 기능

`6. 섹터전략 백테스트` 메뉴에 `1-0) 70/30 방공호 통합 백테스트` 섹션을 추가했습니다.

체크박스:
- `70/30 방공호 같이 계산`

기본 구조:
- 시작 원금 1억 기준
- 대장주 엔진 70%
- 방공호 엔진 30%

방공호 방식:
- `월간 듀얼모멘텀 상위2`
- `정적 50/30/20`
- `현금/단기채만`

기본 후보:
- KODEX200
- KOSDAQ150
- NASDAQ100 ETF
- GOLD ETF
- DOLLAR ETF
- CASH

## 출력 CSV

전체 묶음 다운로드에 아래 파일이 추가됩니다.

- `magic_split_bunker_7030_summary_YYYY-MM-DD.csv`
- `magic_split_bunker_7030_daily_YYYY-MM-DD.csv`

## 해석 주의

대장주 엔진은 현재 섹터전략 백테스트 총자산 곡선을 입력 비중만큼 스케일합니다. 목적은 대장주 시스템 자체를 과최적화하는 것이 아니라, 2020년 같은 폭락장에 전체 계좌 MDD가 얼마나 완충되는지 확인하는 것입니다.
