# v78 T100 LIVE DASHBOARD PANEL

## 수정 내용

7-1. T100 HYBRID 1↔3 단순 운용모드에 **현재 상황 통합판**을 추가했습니다.

통합판에서 바로 확인할 수 있는 항목:

- 현재 누적 투입원금
- 어제 평가금액
- 오늘 평가금액
- 오늘 판정기준
- 1일 변동률
- 하루 -5% 방어 여부
- 5일 누적 변동률
- 5일 -6% 방어 여부
- 최종 방어신호 ON/OFF
- 다음 거래일 행동 안내

## 방어판정 계산

오늘 판정기준금액 = 어제 T100 평가금액 + 오늘 T100 실제 추가매수액 - 오늘 T100 현금화매도액

1일 변동률 = 오늘 T100 평가금액 / 오늘 판정기준금액 - 1

하루 -5% 이하이면 방어 ON입니다.
5일 기록이 충분하면 최근 5개 1일 변동률 누적으로 -6% 이하인지도 같이 확인합니다.
