# MAGIC V27 OPERATION FEAR BUY WORDING FIX

`v27_OPERATION_TOP50_GUIDE_SYNC_20260619` 기반 화면/안내 정리 패치입니다.

## 수정 이유

운영판단기에서는 회수모드라서 매수 금지처럼 보이고, 보유차수 판단기에서는 공포매수 후보가 뜨면서 사용자가 헷갈릴 수 있었습니다.

정확한 원칙은 다음입니다.

- 운영판단기가 금지하는 것: TOP50 신규매수, 새 종목 확장, 15만원→20만원/25만원 금액확장
- 보유차수 판단기가 허용하는 것: 기존 보유종목 중 A급 기준도달 공포매수
- B급 도달검토는 즉시매수가 아니라 수동검토
- 기준가대기는 아직 매수 아님

## 수정 내용

- 운영판단기 결과에 `오늘 결론` 안내 추가
- `공포매수` 문구를 `보유 공포매수`로 변경
- 회수모드 문구를 `신규매수 금지 / 금액확장 잠금 / 보유 공포매수 조건부 허용`으로 정리
- 보유차수 판단기 상단에 `운영판단기가 막는 것은 신규매수와 금액확장` 안내 추가
- TOP50 안내에 `공포매수는 보유차수 판단기 우선` 문구 명확화
