# Magic Split v27 Sector Flow Tables 20260629

이번 버전은 `섹터 순환매 판단기` 안에서 거래대금 움직임을 **섹터요약 / 순환매이동 / 대표주상세 / 발빼기후보** 4개 표로 나눠 보게 만든 버전입니다.

## 핵심 변경

- 앱 버전: `v27_SECTOR_FLOW_TABLES_20260629`
- 기존 TOP50 / 보유종목 판단기 / 운영판단기 구조는 유지
- 새 메뉴는 그대로 `5. 섹터 순환매 판단기`
- 기본 대표주 유니버스: 22개 섹터 / 141개 종목
- `조사 기반 섹터 대표주 채우기` 버튼 유지
- `현재 대장/2등/회전형 자동 재분류` 옵션 추가


## 화면 구조

`섹터 순환매/발빼기 판단 실행` 후 결과가 아래 4개 탭으로 나뉩니다.

1. `섹터요약`
   - 섹터별 오늘 거래대금, 전일 거래대금, 5일 평균 거래대금
   - 쏠림점수, 전일쏠림점수, 5일평균쏠림점수, 쏠림변화
   - 현재대장주, 매수허용, 발빼기신호

2. `순환매이동`
   - 쏠림변화와 5일대비 거래대금 증감률 기준으로 유입/이탈 섹터 분리
   - 낮았다가 올라오는 섹터와 이미 쏠린 뒤 둔화되는 섹터를 따로 확인

3. `대표주상세`
   - 기준역할과 현재역할 비교
   - 섹터내 거래대금 순위, 상승률 순위, 역할변경신호 확인

4. `발빼기후보`
   - 섹터 기준 발빼기 후보
   - 종목 기준 신규추격금지/발빼기 후보

## 역할 구조

기존에는 `섹터역할` 하나만 있었습니다.

이번 버전은 아래처럼 나눕니다.

- `기준역할`: 사용자가 등록한 원래 대장주 / 2등대표주 / 회전형중형주
- `현재역할`: 실행 시점 거래대금 순위로 자동 재분류한 현재대장 / 현재2등대표 / 현재회전형 / 감시
- `역할변경신호`: 대장유지, 기준대장둔화, 현재대장승격후보, 회전부상, 대표주둔화 등
- `섹터내거래대금순위`: 해당 섹터 안에서 오늘 거래대금 순위
- `섹터내상승률순위`: 해당 섹터 안에서 5일 수익률 순위
- `역할점수`: 거래대금 비중 + 거래대금 증가율 + 5일 수익률을 섞은 현재 역할 참고 점수

## 판단 목적

이 메뉴는 종목 추천기가 아니라 섹터 자금 이동 감시용입니다.

앱은 대표주들의 거래대금으로 아래를 계산합니다.

- 특정 섹터 쏠림점수
- 5일 평균 대비 거래대금 유입
- 기준 대장주가 둔화되는지
- 2등 대표주가 현재대장 후보로 올라오는지
- 회전형 종목이 새로 부상하는지
- 고점권 신규추격금지
- 쏠림 후 둔화/이탈 시 발빼기 검토

## 사용 방법

1. 앱 실행
2. `5. 섹터 순환매 판단기` 이동
3. `조사 기반 섹터 대표주 채우기` 클릭
4. `현재 대장/2등/회전형 자동 재분류` 체크
5. `섹터 순환매/발빼기 판단 실행` 클릭

## 주의

- `기준역할`은 고정 감시축입니다. 매일 바꾸지 않습니다.
- `현재역할`은 거래대금 순위로 매번 바뀔 수 있습니다.
- 회전형은 순환매 감지용이라 가장 자주 바뀌는 것이 정상입니다.
- C급/요양원/보유차수 판단 로직은 기존 메뉴에서 그대로 처리합니다.
