# v27_SECTOR_FAST_REENTRY_BACKTEST_FIX_20260629

## 수정 내용

이 버전은 빠른 섹터 백테스터에서 모든 일자가 `계산실패`로 떨어지는 문제를 수정한 긴급 패치입니다.

### 원인

pandas 3.x 계열에서 `float64`로 생성된 빈 판정 컬럼에 `대기`, `사기`, `추가매수금지`, `전량회수` 같은 문자열을 넣으면서 아래 오류가 발생했습니다.

```text
Invalid value '대기' for dtype 'float64'
```

대표주 흐름표 컬럼 중 `오늘행동`, `행동사유`, `대장주예외허용`, `현재역할`, `역할변경신호` 등이 처음에는 NaN으로 생성되어 float 컬럼이 된 것이 문제였습니다.

### 조치

- 섹터 흐름/대표주 판정용 문자열 컬럼을 object 타입으로 강제 변환
- `오늘행동` 문자열 판정 입력 전 dtype 보정
- 빠른 백테스터의 `_build_sector_rotation_flow_from_map()`에도 동일 적용
- 일반 섹터 순환매 판단 함수에도 동일 적용

### 유지된 기능

- 5. 섹터 순환매 판단기
- 6. 섹터전략 백테스트
- 빠른 가격맵 로딩
- 대장주 일부팔기 후 재진입대기/재진입허용
- 대장주 +8% 절반익절 / +12% 잔여익절
- 2등대표주 +5% 익절
- 회전형 +3% 익절

## 권장 테스트 설정

```text
동시 진입 섹터 수: 1
회전형 포함: OFF
```
