# v27_SECTOR_BACKTEST_PROFIT_UP_ROLE_GUARD_20260629

수익확대형 백테스터에 역할보호 손실축소를 추가한 버전입니다.

## 핵심

- 15% 수익형/PROFIT_UP 금액 구조 유지
  - 한 섹터 최대금액 3,000만
  - 동시 진입 섹터 2개
  - 대장주 1차 1,000만 / 2차 500만 / 3차 300만
  - 2등대표주 1차 500만 / 2차 300만
  - 회전형 기본 OFF
- 손실축소는 가격만 보고 실행하지 않음
  - 대장주가 현재대장 유지 중이면 -8%라도 일부회수하지 않음
  - 대장주가 현재대장에서 밀렸을 때만 역할이탈 손실축소 일부회수
  - 2등대표주가 현재대장/현재2등대표 유지 중이면 -5%라도 전량회수하지 않음
  - 2등대표주가 역할에서 밀렸을 때만 역할이탈 손실축소 전량회수
- -6% 경고는 매도하지 않고 추가매수금지만 걸어 무리한 물타기를 막음
- 역할이 다시 회복되면 추가매수금지 상태를 해제하고 정상 보유로 복귀 가능
- 전체 CSV 묶음 다운로드 유지

## 검증 포인트

- 기존 15.28% 버전보다 수익률이 올라가는지
- PROFIT_UP에서 손실축소가 너무 빨라 수익을 깎던 문제가 줄었는지
- 최대낙폭이 -5% 안쪽인지
- 최저현금이 5,000만 원 밑으로 내려가지 않는지
