# v72 T100 A분할 차수 순차진입 수정

## 수정 이유
v70/v71 A분할 백테스트에서는 월초 기준가 대비 -5%, -10%, -15%를 한 번에 검사했습니다.
그래서 하루에 가격이 크게 빠지면 같은 날 3차와 4차가 동시에 들어가는 현상이 생겼습니다.

예: 2026-06-08 KODEX200에서 추가3차와 추가4차가 같은 가격/같은 날에 동시에 발생.

## v72 수정
- 추가매수 기준을 `월초 기준가 누적 하락률`이 아니라 `직전 활성차수 매수가 대비 -5%`로 변경했습니다.
- 한 거래일에 한 종목당 최대 1개 차수만 추가됩니다.
- 1차만 남고 추가차수가 모두 청산되면 새 1차 사이클로 보고 추가차수 기록을 리셋합니다.
- 기존 v68 기능(2010 장기검증, Turbo 빠른 실행, 확장형 SP500+BOND, 7-1 운용모드)은 유지했습니다.

## 메뉴
- 6. 섹터전략 백테스트: 기존 Turbo/T100 백테스트
- 7-1. T100 하이브리드 운용모드
- 10. T100 A분할 백테스트

## A분할 기본값
- 종목당 1차 30,000,000원
- 직전차수 대비 -5%마다 5,000,000원 추가
- 최대 6회 추가
- 차수별 +5% 익절
- 1차 익절 후 재진입 ON
- 하루 최대 1차수 추가
