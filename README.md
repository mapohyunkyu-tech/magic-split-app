# MAGIC v27 HOLDING ACTION FOCUS UI

보유차수 판단기 화면 가독성/실전 액션 중심 패치입니다.

## 핵심 변경

- 보유차수 판단표에 `핵심행동`, `현재장판단` 컬럼 추가
- 화면 상단에 아래 3개 요약표 표시
  - 🟢 오늘매수가능
  - 🟡 기준가대기
  - 🔵 회수/익절검토
- FDR 현재가 지연 대비용으로 HTS/증권앱에서 볼 기준가를 `현재장판단`에 짧게 표시
- 전체 판단표는 그대로 유지
- 저장된 보유차수는 전부 유지하고, 판단은 종목별 최고차수만 표시
- 종목/코드 고정 컬럼 유지

## 사용법

보유차수 판단기를 실행한 뒤 전체 표를 먼저 보지 말고, 위쪽 요약표 3개를 먼저 확인하세요.

- 오늘매수가능: 앱 기준 이미 도달한 후보
- 기준가대기: HTS 현재가가 추가매수기준가 이하로 내려오면 확인할 후보
- 회수/익절검토: 수익권 또는 본전권 회수 후보
