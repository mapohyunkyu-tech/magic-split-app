# v77 T100 운용기록 전체 초기화 복구 방지 수정

## 수정 이유
v74 이후 T100 운용기록을 사용자 홈 폴더 `magic_split_data/t100_hybrid_live_history.csv`에 저장하도록 바꿨습니다.  
그런데 전체 초기화로 홈 폴더 기록을 삭제해도, 앱 폴더에 남아 있던 구버전 `t100_hybrid_live_history.csv`가 다음 실행 때 다시 자동 복사되어 기록이 계속 남는 문제가 있었습니다.

## v77 수정 내용
- `운용기록 전체 초기화` 클릭 시 홈 폴더 기록 삭제
- 앱 폴더의 구버전 기록 파일도 삭제 시도
- 삭제 후 `.t100_hybrid_history_cleared` 마커 생성
- 마커가 있으면 구버전 기록 자동복사 차단
- 새 운용기록을 저장하거나 백업 CSV를 복원하면 마커 자동 해제
- 전체 초기화 후 입력칸도 기본값으로 되돌림

## 사용 방법
1. `7-1. T100 하이브리드 운용모드`로 이동
2. `운용기록 전체 초기화` 클릭
3. 최근 저장기록이 사라지는지 확인
4. 새 기준일을 다시 저장

## 유지 기능
- v68의 2010 장기검증 / Turbo 빠른실행 유지
- v75의 입출금 보정 방어판정 유지
- v76의 Streamlit 저장 오류 수정 유지
