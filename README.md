# Magic Split v27 - Dual Market Backtest Filter (2026-06-30)

## 핵심 추가

`6. 섹터전략 백테스트`에 시장 붕괴 필터를 2단 구조로 추가했습니다.

### 1) 시장붕괴 필터(KODEX200)
- 전체시장 붕괴 차단용
- 신규매수/추가매수/재진입매수만 차단
- 매도/일부팔기/전량회수는 그대로 실행

옵션:
- OFF
- 코로나급만 차단: 20일 -12% 이하 또는 10일 -8% 이하 또는 당일 -4% 이하
- 강한차단
- 초강한차단

### 2) 성장주 보조필터(KOSDAQ150)
- KODEX 코스닥150 ETF(229200) 기준
- KODEX200은 버티는데 코스닥/성장주만 무너지는 장을 보조 차단
- 적용 섹터: 바이오, 로봇, 엔터/게임, 인터넷/소프트웨어, 2차전지, 반도체, PCB, 화장품
- 해당 섹터의 신규매수/추가매수/재진입매수만 차단
- 매도/일부팔기/전량회수는 그대로 실행

옵션:
- OFF
- 코스닥급락만 차단: KOSDAQ150 20일 -15% 이하 또는 10일 -10% 이하 또는 당일 -4.5% 이하
- 강한성장주차단
- 초강한성장주차단
