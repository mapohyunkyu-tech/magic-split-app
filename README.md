# v27_SECTOR_LIVE_OPERATION_BOARD_EMERGENCY_RISK_20260629

## 핵심 업데이트

기존 실전 운영판 RULE_FIX 버전에 `긴급위험모드`를 추가했습니다.

목적은 악재 날짜를 결과 보고 빼는 것이 아니라, 백테스트와 실전에서 **사전에 정한 지수 쇼크 룰**이 발동하면 신규/추가매수만 차단하고 매도/축소 신호는 그대로 실행하는 것입니다.

## 백테스트 긴급위험모드

메뉴: `6. 섹터전략 백테스트` → `4-3) 긴급위험모드`

옵션:

- OFF
- 급락일만 차단
- 지수쇼크 차단
- 강한위험 차단
- 초강한위험 차단

발동 시 처리:

- `사기` 신규/추가매수 예산 0
- `대기`, `신규금지`, `추가매수금지` 유지
- `일부팔기`, `전량회수` 매도 신호는 그대로 실행

추천 비교 순서:

1. OFF
2. 지수쇼크 차단
3. 강한위험 차단

## 실전 운영판 긴급위험모드

메뉴: `7. 실전 운영판`

옵션:

- OFF
- 수동ON

수동ON 발동 시:

- `사라` → `사지마`
- `대기` → `대기`
- `줄여라` → `줄여라`
- `팔아라` → `팔아라`

즉 악재/전쟁/정책발언/외국인 패닉 같은 날에는 신규매수만 죽이고, 기존 보유 축소/회수 신호는 유지합니다.

## 유지된 기본값

- 수익확대프리셋: C 조합공격
- 장세매수필터: MDD10 상승강화 + 횡보차단
- 섹터자동가중: 표준형
- MDD브레이크: 완만브레이크 또는 표준브레이크
- 손실쿨다운: 완만쿨다운
- 회전형: OFF
- 대장주 익절: +10% / +18%
