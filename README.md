# MAGIC V27 OPERATION TOP50 GUIDE SYNC

`v27_B_GRADE_REACHED_REVIEW_20260619` 기반 안내/연동 정리 패치입니다.

## 수정 이유

공포매수 탄창제, 금액확장 잠금, A/B/C 자동등급, B급 도달검토 기능은 보유차수 판단기 중심으로 들어갔지만, 운영판단기와 TOP50 화면 문구가 일부 구버전처럼 보여 헷갈릴 수 있었습니다.

특히 회수모드에서 기존 안내가 `신규매수 금지, 예수금 회복` 쪽으로만 보여서, 사용자의 핵심 철학인 `신규매수/금액확장은 금지하되 기존 보유 A급 기준도달 공포매수는 허용`이 화면에 명확히 드러나지 않았습니다.

## 수정 내용

- 앱 버전명 갱신
  - `v27_OPERATION_TOP50_GUIDE_SYNC_20260619`
- 운영판단기 문구 정리
  - 신규매수 금지
  - 금액확장 금지
  - 보유 A급 기준도달분은 공포매수 탄창제 한도 내 예외 허용
- 제한회복모드/제한매수모드/손실주의 정상운용 안내 문구 정리
- TOP50 화면 안내 추가
  - TOP50은 신규 후보 참고용
  - 회수/공포매수 구간에서는 보유차수 판단기의 A급/B급/기준가 도달표 우선
- 보유차수 판단기 캡션 정리
  - 운영판단기 오늘값/테스트값 연동
  - 공포매수 탄창, 금액확장 잠금, A/B/C 자동등급, 기준가 도달 여부 계산 안내
- 도움말 문구 정리
  - 오늘매수가능 / B급 도달검토 / 기준가대기 / 회수익절검토 구조 반영

## 유지된 기존 기능

- 간편보유 입력
- 정밀차수 우선 + 간편보유 차수 보정
- 마지막 차수 1줄 판단
- 추가매수 기준가 / 현재장도달시
- 공포매수 탄창제
- A/B/C 자동등급
- B급 도달검토
- C등급 기준가대기 제외
- 종목/코드 고정 표시
