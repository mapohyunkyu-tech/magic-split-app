# IDIO 300 v3 Champion

비체계적 과락 반등 포트 전략 Streamlit 백테스터입니다.

## 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 파일

- `app.py`: Streamlit 앱
- `requirements.txt`: 설치 패키지 목록

## 주의

자동수집(pykrx) 모드는 무료 데이터 기반 1차 검증용입니다. 정밀 검증에는 상장폐지 종목, 관리종목/거래정지 이력, 과거 섹터 분류, 과거 시가총액 등을 포함한 별도 DB가 필요합니다.
