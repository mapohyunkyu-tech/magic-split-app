# MAGIC v27 LIVE TRIGGER PRICE HINT

보유차수 판단기 보정판입니다.

## 핵심 수정

FDR 현재가가 실제 증권앱/HTS 현재가보다 늦게 들어오는 경우를 대비해, 보유차수 판단표에 다음 컬럼을 추가했습니다.

- `추가매수기준가`: 현재 차수의 다음 차수 매수 기준 가격
- `현재장도달시`: HTS/증권앱 현재가가 기준가 이하로 내려오면 매수가능 후보인지 표시

예시:

```text
DB하이텍 2차
진입단가 174,200
2차→3차 기준 -11.11%
추가매수기준가 154,842원
현재장도달시 HTS 154,842원 이하 도달시 매수가능후보
```

즉 FDR 가격상 미도달이어도, 실제 현재장에서 기준가를 터치하면 바로 확인할 수 있습니다.

## 주의

`현재장도달시`는 가격 지연 보조 안내입니다.
과열, 당일급락, 회수모드, 점수 부족 등 다른 금지사유가 있으면 자동으로 매수가능후보 문구를 띄우지 않습니다.
