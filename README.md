# v79_T100_PREFILL_LATEST_SAVED_INPUTS

## 수정 내용

`7-1. T100 HYBRID 1↔3 단순 운용모드`에서 최근 저장된 운용기록을 입력칸에 자동 반영하도록 수정했습니다.

### 핵심 변경

- 새 ZIP/app.py로 갈아도 최근 저장기록이 있으면 입력칸 기본값에 자동 반영
- `T100 누적 투입원금 · 오늘 입출금 전` = 최근 저장기록의 `투입원금`
- `오늘 장마감 T100 평가금액` = 최근 저장기록의 `당일평가금액`
- `계좌 예수금 · 참고` = 최근 저장기록의 `계좌예수금메모`
- `최근 저장값 입력칸 불러오기` 버튼 추가
- 저장 후 다음 화면에서도 투입원금과 오늘 평가금액 입력칸이 방금 저장값으로 유지됨

## 사용법

1. 앱 실행
2. `7-1. T100 HYBRID 1↔3 단순 운용모드` 이동
3. 최근 저장기록이 있으면 상단에 최근 저장값이 표시됨
4. 입력칸이 안 맞으면 `최근 저장값 입력칸 불러오기` 클릭
5. 오늘 장마감 평가금액만 수정 후 저장

## 주의

- 투입원금은 추가입금/인출할 때만 변경합니다.
- 오늘 평가금액은 장마감 후 실제 T100 ETF 평가금액으로 수정합니다.
- 방어판정은 투입원금 기준이 아니라 `오늘 평가금액 ÷ 오늘 판정기준 - 1`로 계산합니다.
