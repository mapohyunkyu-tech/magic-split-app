# MAGIC V27 SIMPLE HOLDING DELETE FIX

`v27_FULL_EXPLAIN_SAFETY_UPDATE_20260619` 기반 보완판입니다.

## 수정 이유

익절/전량매도한 종목을 `보유간편`에 저장한 뒤, 화면에서 개별 삭제하기가 어려웠습니다.
기존에는 간편 보유 전체 삭제는 가능했지만, 특정 종목만 빼는 흐름이 불편했습니다.

## 수정 내용

- `보유간편` 표 왼쪽에 `삭제` 체크 컬럼 추가
- 체크 후 `선택한 간편 보유 삭제` 버튼으로 일부 종목만 삭제 가능
- `익절/매도 종목명으로 바로 삭제` 입력창 추가
  - 예: `삼성E&A, 서진시스템`
- 종목명 삭제 시 코드로 매칭해서 `보유간편`에서 제거
- 정밀 보유차수는 건드리지 않음
- 빈 행/삭제행이 Google Sheets에 이상한 코드로 남지 않도록 저장 로직 보강

## 사용법

1. 보유차수 판단기 → 간편 보유 입력 영역을 엽니다.
2. 저장된 간편 보유 표에서 익절/매도 종목의 `삭제`를 체크합니다.
3. `선택한 간편 보유 삭제`를 누릅니다.

또는 종목명 입력창에 아래처럼 입력 후 삭제할 수 있습니다.

```text
삼성E&A, 서진시스템
```

## 주의

- 이 기능은 `보유간편`만 삭제합니다.
- 정밀 보유차수에 저장된 차수는 별도의 보유차수 삭제 기능으로 지워야 합니다.
