# v27_SECTOR_LIVE_OPERATION_BOARD_20260629

## 핵심

백테스트 실험판을 실전 운영판으로 정리한 버전입니다.

## 기본 실전 세팅

- 수익확대프리셋: C 조합공격
- 장세매수필터: MDD10 상승강화 + 횡보차단
- 섹터자동가중: 표준형
- MDD브레이크: 완만브레이크 기본, 균형형은 표준브레이크
- 손실쿨다운: 완만쿨다운
- 대장주: 1차 1,300만 / 2차 500만 / 3차 300만
- 2등대표주: 1차 700만 / 2차 300만
- 대장주 익절: +10% / +18%
- 2등대표주 익절: +5%
- 증액투자: ON
- 회전형: OFF

## 추가 메뉴

`7. 실전 운영판` 메뉴를 추가했습니다.

- 최종 후보 세팅표
- 오늘 섹터 행동판
- 대표주 상세 행동
- 실전 운영판 CSV 묶음 다운로드
