# MAGIC V27 FEAR BUY EXCEPTION FIX

`v27_FEAR_BUY_ALLOWED_EXPANSION_LOCK_20260619` 기반 보정 패치입니다.

## 수정 이유

실행 화면에서 운영모드가 `회수모드`가 되면:

- 공포매수상태가 `공포매수보류`로 표시되고
- 일일 매수상한이 0원
- 추가매수 가능개수가 0개
- A급 기준도달 보유종목도 최종 매수가능으로 올라오지 않는 문제가 있었습니다.

이건 사용자의 원래 철학인 `확장은 잠그되 공포매수는 기존 보유 A/B급 기준도달분만 허용`과 어긋납니다.

## 수정 내용

- `회수모드`에서도 공포매수상태를 `공포매수허용`으로 표시
- `회수모드`에서는 신규매수와 금액확장은 계속 금지
- 단, 보유 A급 + 마지막 차수 기준가 도달 종목은 기본금액 기준 1~2개 예외 허용
- `강한 회수모드` 또는 예수금 절대방어선 미만일 때만 현금방어 우선
- 보유차수 판단표의 최종 `오늘추가매수`도 회수모드 A급 기준도달분을 반영

## 운영 해석

- 신규매수: 금지
- 금액확장: 잠금
- 공포매수: 허용
- 허용 대상: 기존 보유종목 중 자동 A등급 + 마지막 차수 기준가 도달
- 금액: 확장금액이 아니라 기본금액 유지
