# MAGIC v27 HOLDING LATEST LOT VIEW

v27_HOLDING_LOT_TRIGGER_FIX 기반 안정 패치입니다.

## 수정 내용

- 보유차수 저장 데이터는 1차/2차/3차 등 모든 입력 행을 그대로 유지합니다.
- 보유차수 판단표는 종목별 현재 보유 최고차수 1줄만 표시합니다.
  - 예: 1차와 2차를 보유 중이면 판단표에는 2차만 표시
  - 2차의 현재 손익률로 3차 매수 기준(-11.11%) 도달 여부를 판단
- 낮은 차수 행은 판단표에서 숨겨서 불필요한 행을 줄였습니다.
- 판단표 컬럼 순서를 조정했습니다.
  - 순위 | 종목 | 코드 | 오늘추가매수 | 보유판정 | 판정사유 ...
- 종목총수량, 종목평균단가_자동, 종목전체평가손익_자동, 종목전체손익률_자동은 숨겨진 낮은 차수까지 포함한 전체 저장차수 기준으로 계산합니다.

## 차수 기준

- 1차 → 2차: -10.00%
- 2차 → 3차: -11.11%
- 3차 → 4차: -12.50%
- 4차 → 5차: -14.29%
- 5차 → 6차: -16.67%
