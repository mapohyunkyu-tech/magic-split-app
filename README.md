# MAGIC v27 JUDGE REASON TEXT FIX

버전: `v27_JUDGE_REASON_TEXT_FIX_20260616`

## 수정 목적

TOP50 결과에서 `최종판정=금지`, `오늘매수=금지`인데 판정사유가 `오늘매수 대기 / 당일급등`처럼 표시되어 헷갈리는 문제를 수정했습니다.

## 수정 내용

- `judge_final_entry()` 판정 문구 정리
- 금지 판정일 때는 예산/순위성 문구인 `오늘매수 대기`를 판정사유에서 제거
- 대기 판정일 때는 기존처럼 `오늘매수 대기` 표시 유지
- 진입가능 판정일 때는 `조건통과` 표시 유지
- 섹터/테마/미분류 자동학습 로직은 건드리지 않음

## 예시

기존:

```text
최종판정: 금지
오늘매수: 금지
판정사유: 오늘매수 대기 / 당일급등 14.23%
```

수정 후:

```text
최종판정: 금지
오늘매수: 금지
판정사유: 당일급등 14.23%
```

기존:

```text
최종판정: 대기
오늘매수: 대기
판정사유: 오늘매수 대기
```

수정 후:

```text
최종판정: 대기
```
