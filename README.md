# MAGIC V27 ADD BUY AT CAP FIX

버전: v27_ADD_BUY_AT_CAP_FIX_20260617

## 수정 이유

기존 `v27_HOLDING_LATEST_LOT_VIEW`에서는 15만원/120종목 구간에서 총 보유종목수가 120개가 되는 순간 `강한 회수모드`로 전환되어, 보유종목이 다음 차수 기준(-10%, -11.11%, -12.50%, -14.29%, -16.67%)에 도달해도 추가매수가 전부 막히는 문제가 있었습니다.

## 수정 내용

- 보유종목수 = 목표종목수 또는 최대종목수 도달
  - 신규매수 중단
  - 보유종목 기준도달 추가매수는 제한 허용
  - 운영모드: 제한회복모드

- 보유종목수 > 최대종목수 초과
  - 회수모드
  - 신규/추가매수 중단
  - 회수 우선

## 기대 결과 예시

15만원/120종목 구간에서 총 보유종목수 120개인 경우:

- 기존: 강한 회수모드 / 추가매수 가능개수 0개
- 수정 후: 제한회복모드 / 신규매수 0개 / 추가매수 가능개수 제한 허용

즉, 신규 종목은 더 안 사고, 이미 보유한 종목이 마지막 차수 기준에 도달했을 때만 추가매수 후보로 표시됩니다.
