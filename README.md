# v83 T100 눌림목 한방 보조계좌 추가

기준 베이스: v82 미국 ETF 데이터로더 디버그 + v79 T100 실전운용판 복구 상태.

## 유지된 기능
- 7-1 T100 HYBRID 1↔3 단순 운용모드
- 현재 상황 통합판
- 최근 저장값 자동 불러오기
- 투입원금 / 어제 평가금 / 오늘 평가금 / 하루 -5% / 5일 -6% 표시
- 운용기록 고정 저장/백업/초기화
- 9. 미국 ETF T100 백테스트

## v83 추가 기능
7-1 메뉴에 `4-2) 눌림목 한방 보조계좌`를 추가했습니다.

### 기본 룰
- 메인 T100은 건드리지 않음
- 방어 OFF일 때만 신규진입
- 대상은 이번 달 T100 선택 ETF만 사용
- 최근고점 대비 -3% ~ -7% 눌림
- 20일선 위 또는 20일선 근처
- 전일 대비 반등 확인
- 익절 +4%, 손절 -3%, 최대 5거래일 보유
- 눌림목 전용자금 안에서만 진입

## 실행
```bash
streamlit run app.py
```
