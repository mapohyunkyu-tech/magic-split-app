# v67 T100 확장형 SP500+BOND 백테스트

## 목적
기존 T100 후보군에 `SP500`과 `BOND`를 추가한 확장형을 기존 기본형과 비교한다.

## 기존 기본형 후보
- KODEX200
- KOSDAQ150
- NASDAQ100
- GOLD
- DOLLAR
- CASH

## v67 확장형 후보
- KODEX200
- KOSDAQ150
- NASDAQ100
- SP500
- GOLD
- DOLLAR
- BOND
- CASH

## 기본 코드 후보
- KODEX200: 069500
- KOSDAQ150: 229200 / 233740
- NASDAQ100: 133690 / 379810 / 381170
- SP500: 360750 / 379800 / 143850
- GOLD: 411060 / 132030 / 319640
- DOLLAR: 261240 / 138230
- BOND: 114260 / 152380

## 실행 위치
1. `streamlit run app.py`
2. `6. 섹터전략 백테스트`
3. `1-0) 073 / 064 / CTA 방공호+부스터 통합 백테스트`
4. 방공호 방식에서 아래 둘을 비교
   - `T10/T100 Turbo 공격ETF 100% NO CTA`
   - `T10/T100 Turbo 확장형 SP500+BOND`

## 리밸런싱 규칙
- 월 1회
- 후보군 중 모멘텀 상위 2개 선택
- 선택된 2개를 50:50 보유
- 기본형은 기존 후보만 사용
- 확장형은 SP500/BOND까지 후보에 포함

## 주의
확장형이 무조건 더 좋다는 뜻은 아니다. 후보군이 늘면 기회가 늘 수 있지만, 약한 자산을 고점에서 잡는 경우도 생길 수 있으므로 기본형과 비교 후 선택한다.
