# 홀딩스캐너 백테스트 v1

## 파일
- app.py
- requirements.txt
- runtime.txt

## 기능
- FinanceDataReader 일봉으로 KRX 종목 백테스트
- 매월 스캔 → 상위 N개 매수 가정 → 3/6/12개월 뒤 수익률 확인
- 차트 조건: 눌림목, 기준봉, 돌파, 저점상향, 20/60/120일선, 거래대금
- 선택적으로 재무점수 CSV를 업로드해 결합 가능

## 재무 CSV 예시
컬럼 중 하나 이상 사용:
- 종목코드, 종목명, 재무총점
- 또는 종목코드, 종목명, 재무점수, 저평가점수, 실적점수, 재무안정점수

## 주의
v1은 검증용입니다. 수수료/세금/슬리피지/재무데이터 시차는 단순화되어 있습니다.
