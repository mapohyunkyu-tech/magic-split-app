## 핵심 변경

- 대장주 재진입 기본값을 `1차 100%`에서 `1차 50%`로 변경
- 전량회수까지 기다리기 전에 손실축소 룰 추가
- 회전형 기본 OFF 유지
- 전체 CSV 묶음 다운로드 유지
- 기간 프리셋 유지: 60 / 120 / 240 / 480 / 사용자지정

## 손실축소 기본값

- 대장주 손실 -6% 이하 + 신호 약화 또는 현재대장 이탈
  - 매도는 하지 않고 `추가매수금지` 상태로 변경
- 대장주 손실 -8% 이하 + 신호 약화 또는 현재대장 이탈
  - 절반 일부회수
  - 남은 물량은 `재진입대기`
- 2등대표주 손실 -5% 이하 + 신호 약화 또는 현재2등대표 이탈
  - 전량회수

## 재진입 기본값

- 대장주만 재진입 허용
- 발빼기 후 섹터가 다시 `사기`로 회복되고 현재역할이 `현재대장`이면 1차의 50%만 재진입
- 2등대표주/회전형은 발빼기 후 자동 재진입하지 않음
