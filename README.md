# v27_SECTOR_BACKTEST_REGIME_BUY_FILTER_20260629

섹터형 매직스플릿 백테스터에 **장세별 매수차단/축소 테스트**를 추가한 버전입니다.

## 유지한 기본 구조

- 핵심만 / 전체 대표주 검증 선택
- 증액투자 모드 ON/OFF
- 동시 2섹터 기본
- 대장주 1,000만 / 500만 / 300만
- 2등대표주 500만 / 300만
- 회전형 기본 OFF
- 현재대장/현재2등 유지 중이면 손실축소 매도 금지
- 역할에서 밀렸을 때만 손실축소
- 월별결과 / 장세별결과 / 전체 CSV 묶음 다운로드 유지

## 새로 추가한 기능

### 장세별 매수필터

백테스트 메뉴에 `4) 장세별 매수차단/축소 설정`을 추가했습니다.

옵션:

1. OFF
   - 기존과 동일하게 섹터 신호만으로 매수합니다.

2. 하락/급락 차단
   - 하락장, 급락일에는 신규/추가매수를 차단합니다.
   - 상승장, 횡보장, 조정장은 기존 금액대로 매수합니다.

3. 횡보축소 + 하락차단
   - 강한상승장: 1.2배
   - 상승장: 1.0배
   - 횡보장: 0.5배, 대장주만 허용
   - 조정장: 0.5배, 대장주만 허용
   - 하락장: 0배
   - 급락일: 0배
   - 횡보/조정에서는 2등대표주·회전형 신규/추가매수를 차단합니다.

4. 상승장만 매수
   - 강한상승장/상승장에만 매수합니다.
   - 비교용 보수 모드입니다.

5. 사용자지정
   - 강한상승장, 상승장, 횡보장, 조정장, 하락장, 급락일 배율을 직접 입력합니다.

## 추가 CSV 컬럼

일별 자금흐름 CSV에 아래 컬럼이 추가됩니다.

- 장세
- 장세매수배율
- 장세매수필터

설정 CSV에도 장세매수필터와 장세별 배율이 저장됩니다.

## 테스트 추천 순서

1. 기존 베스트 재확인
   - 증액투자 ON
   - 핵심만
   - 회전형 OFF
   - 장세매수필터 OFF
   - 480/720거래일

2. 하락/급락 차단
   - 수익률이 유지되는지
   - MDD가 줄어드는지 확인

3. 횡보축소 + 하락차단
   - 횡보장 손실과 급락일 손실이 줄어드는지 확인
   - 수익률이 너무 깎이면 폐기

4. 상승장만 매수
   - 비교용으로만 확인
