# v27_SECTOR_BACKTEST_COMPOUND_MODE_20260629

## 기준
- 직전 베스트 버전 `v27_SECTOR_BACKTEST_PROFIT_UP_ROLE_GUARD_20260629` 기반
- 수익확대형 금액 유지
  - 한 섹터 최대금액: 3,000만
  - 대장주: 1,000만 / 500만 / 300만
  - 2등대표주: 500만 / 300만
  - 동시 진입 섹터: 2개
  - 회전형: 기본 OFF

## 추가 기능
### 1. 증액투자 모드 ON/OFF
- OFF: 기존처럼 고정금액 백테스트
- ON: 현재 총자산 / 초기자금 비율만큼 차수금액과 섹터한도를 자동 확대·축소
- 예시: 초기자금 1억, 대장주 1차 1,000만
  - 총자산 1.2억이면 대장주 1차 약 1,200만
  - 총자산 9,000만이면 대장주 1차 약 900만

### 2. 최저현금 기준 프리셋
- 5천만
- 4천만
- 3천만
- 사용자지정

### 3. CSV 설정값 기록
- settings CSV에 `증액투자모드`, `최저현금기준` 컬럼 추가
- summary CSV에 `증액투자모드` 기록

## 유지된 룰
- 현재대장/현재2등 유지 중에는 손실축소 매도하지 않음
- 역할에서 밀렸을 때만 손실축소
- 대장주 +8% 절반익절, +12% 전량익절
- 2등대표주 +5% 익절
- 회전형 기본 OFF
