# MAGIC v27 THEME CRAWLER READY

v27 섹터신뢰/테마실체 리스크 필터에 **뉴스/공시 자동 점검 준비형**을 추가한 버전입니다.

## 핵심 변경

- TOP50/보유차수 공통 섹터엔진 유지
- 섹터신뢰 낮음 신규매수 제한 유지
- 아이로보틱스 같은 테마실체 확인필요 종목 방어 유지
- `최종판정=금지`인데 `오늘매수=매수가능`으로 남는 버그 방지 유지
- `테마원천`, `테마리스크사유` 컬럼 추가
- 네이버 뉴스 공식 Search API + OpenDART API 키가 있으면 외부 뉴스/공시 신호를 자동 점검 가능

## 외부 원천 우선순위

1. KIND/FDR 업종·주요제품: 기존사업/섹터 분류용
2. 네이버 뉴스 공식 Search API: 테마 뉴스 집중/리스크 키워드 확인용
3. OpenDART API: 최근 90일 공시 제목에서 CB/BW/유상증자/불성실공시/거래정지 등 리스크 확인용
4. 내장 코드맵/키워드맵: API 실패 시 안전망

## Streamlit secrets 선택 설정

외부 뉴스/공시 자동 점검을 TOP50 계산에 반영하려면 secrets에 아래를 추가하세요.

```toml
THEME_CRAWL_ENABLED = "true"
NAVER_CLIENT_ID = "네이버_개발자센터_Client_ID"
NAVER_CLIENT_SECRET = "네이버_개발자센터_Client_Secret"
DART_API_KEY = "OpenDART_API_KEY"
```

키가 없거나 `THEME_CRAWL_ENABLED`가 없으면 앱은 기존 내장 섹터/테마 규칙만 사용하고 정상 작동합니다.

## 주의

- 네이버 종목토론실은 욕/찬티/안티 노이즈가 커서 자동 감점 원천으로 쓰지 않았습니다.
- 뉴스/공시 크롤링은 TOP50 전체 호출량을 늘릴 수 있으므로 기본값은 꺼짐입니다.
- `5. 섹터진단`에서는 특정 종목의 외부 뉴스/공시 신호를 확인할 수 있습니다.
