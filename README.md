장세별 배율을 직접 숫자로 조절하지 않아도 되도록, 백테스터에 **프리셋형 매수차단/축소 모드**를 추가한 버전입니다.

## 핵심 변경

기존 기능은 유지하면서 `6. 섹터전략 백테스트`의 `장세별 매수차단/축소 설정`에 아래 프리셋을 추가했습니다.

### 1. 횡보완전차단 + 조정축소
- 강한상승장: 1.2배
- 상승장: 1.0배
- 횡보장: 0배, 신규/추가매수 차단
- 조정장: 0.5배, 대장주만 허용
- 하락장: 0배
- 급락일: 0배

### 2. 상승공격 + 횡보차단
- 강한상승장: 1.4배
- 상승장: 1.2배
- 횡보장: 0배, 신규/추가매수 차단
- 조정장: 0.5배, 대장주만 허용
- 하락장: 0배
- 급락일: 0배

## 기존 프리셋 유지

- OFF

- 횡보축소 + 하락차단
- 상승장만 매수
- 사용자지정
