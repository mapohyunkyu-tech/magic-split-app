# MAGIC V27 FEAR BUY AMMO REFINE

`v27_FEAR_BUY_AMMO_SYSTEM_20260619` 기반 보완 패치입니다.

## 수정 이유

공포매수 탄창제는 적용됐지만, 실사용 점검에서 두 가지 보완점이 있었습니다.

1. 총 보유종목수와 요양원 종목수를 감안했을 때 판단표 종목 수가 부족한지 화면에서 바로 확인하기 어려움
2. 자동등급 C 종목도 `기준가대기` 상단표에 올라와 실제 후보처럼 보일 수 있음

## 수정 내용

### 1. 보유 입력 누락 점검

보유차수 판단 실행 후 다음 값을 계산합니다.

- 활동예상종목 = 총보유종목수 - 요양원종목수
- 판단표시종목 = 정밀/간편 반영 후 실제 판단표 종목 수
- 판단누락추정 = 활동예상종목 - 판단표시종목

차이가 있으면 화면에 경고로 표시합니다.

### 2. C등급 기준가대기 제거

- A/B급만 `🟡 기준가대기` 상단표에 표시
- C/제외 등급은 기준가에 가까워져도 `⛔ 금지`로 표시
- 현재장판단 문구도 `등급 미달: 기준가 도달해도 매수후보 제외`로 표시

## 유지 기능

- 간편보유 입력
- 정밀차수 + 간편차수보정
- 공포매수 탄창제
- 금액확장 잠금
- 오늘매수가능/기준가대기/회수익절검토 요약표
- 종목/코드 고정 표시
