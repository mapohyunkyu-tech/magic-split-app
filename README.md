# 매직스플릿 v27_SECTOR_ROTATION_FLOW_EXIT_20260629

이번 버전은 기존 TOP50/보유종목 판단기 안에 컬럼을 끼워 넣는 방식이 아니라, 완전히 독립된 `5. 섹터 순환매 판단기` 메뉴를 추가한 버전입니다.

## 핵심 목적

- 섹터별 대장주/2등대표주/회전형대표주를 먼저 정리
- 대표주들의 거래대금을 합산해서 섹터별 자금 쏠림 확인
- 모든 섹터가 균등하면 쏠림점수 5.0으로 표시
- 특정 섹터에 돈이 몰리면 8~10점으로 표시
- 5일 평균 대비 거래대금이 증가하는 섹터는 순환매 유입 후보로 표시
- 거래대금 쏠림 이후 둔화되는 섹터는 발빼기/회수 검토로 표시
- 고점 대표주 추격 매수를 막고, 눌림 또는 다음 순환 섹터를 보게 함
