# v84 T100 하이브리드 개선안 업데이트

## 핵심

v83 실전운용판을 기준으로 다음 규칙을 7-1 메뉴에 반영했습니다.

### 1순위 개선안
- 방어 조건: 1일 -5% 또는 5일 누적 -6%
- 방어 ON: T100 70% / 현금 30%
- 방어 유지: 최소 20거래일
- 공격 복귀: 20거래일 이후 30일선 회복 또는 고점 대비 -5% 이내 회복

### 6310 잠금
- 수익 +50% 도달 시 6310 전환 가능
- 6310 비율: T100 60% / 대기현금 30% / 생활비잠금 10%
- 6310 유지: 최소 60거래일
- 1순위 복귀: 60거래일 이후 회복 조건 충족 시 가능

## 화면 추가
- 현재 상황 통합판 유지
- 원시 방어신호 / 실전 방어신호 분리 표시
- 방어 유지일수 표시
- 30일선 회복 여부 표시
- 고점 대비 낙폭 표시
- 6310 유지일수 / 복귀가능 여부 표시
- 저장기록에 v84 운용상태 컬럼 저장

## 주의

투입원금은 추가입금/인출 때만 바뀝니다.
-5% 방어판정은 투입원금 기준이 아니라 전일 평가금액과 오늘 실제 T100 추가매수/현금화매도 보정 기준으로 계산합니다.
