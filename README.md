# MAGIC v27 HOLDING LOT TRIGGER FIX

버전: `v27_HOLDING_LOT_TRIGGER_FIX_20260617`

## 수정 핵심

보유차수 판단기에서 단순히 `-5% 이상 손실`이면 추가매수 후보로 보이던 문제를 수정했습니다.

이제 추가매수 후보는 매직스플릿 차수별 기준에 도달해야 표시됩니다.

| 현재 차수 | 다음 차수 | 추가매수 기준 |
|---:|---:|---:|
| 1차 | 2차 | -10.00% |
| 2차 | 3차 | -11.11% |
| 3차 | 4차 | -12.50% |
| 4차 | 5차 | -14.29% |
| 5차 | 6차 | -16.67% |

## 화면/CSV 추가 컬럼

보유차수 판단 결과에 아래 컬럼이 추가됩니다.

- `다음차수`
- `추가매수기준률`
- `기준까지남은하락률`
- `기준도달여부`

예를 들어 1차가 -4%라면:

- 추가매수기준률: `10.00`
- 기준까지남은하락률: `6.00`
- 기준도달여부: `미도달`
- 오늘추가매수: `금지`

## 여러 차수 입력 시 처리

한 종목에 1차, 2차를 모두 입력했다면 오늘 추가매수 판단은 현재 보유 중인 최고 차수만 봅니다.

예:

```text
DB하이텍
1, 193300, 1
2, 174200, 1
```

이 경우 1차 행은 `하위차수유지`로 표시되고, 2차가 3차 매수 기준인 `-11.11%`에 도달했는지만 판단합니다.
