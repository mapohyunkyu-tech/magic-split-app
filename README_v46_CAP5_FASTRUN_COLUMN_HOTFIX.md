# Magic Split v46 - CAP5 빠른 실행 컬럼 인식 핫픽스

## 수정 이유
`방공호/Turbo만 빠른 실행`에서 `T100-CAP5-R70-MIN5 현실형`을 선택했을 때 아래 오류가 발생할 수 있었습니다.

```text
방공호/Turbo 빠른 실행 중 오류: CAP5 계산에는 기준일/통합총자산 컬럼이 필요합니다.
```

원인은 빠른 실행 경로에서 생성된 T100 daily 결과의 컬럼명이 환경/모드에 따라 `통합총자산`이 아니라 다른 자산 컬럼명으로 넘어오거나, 날짜가 인덱스로 들어오는 경우를 CAP5 계산기가 충분히 인식하지 못했기 때문입니다.

## 수정 내용
- CAP5 계산 입력 컬럼 자동 인식 강화
- `기준일`, `Date`, `날짜`, `일자`, `거래일` 자동 인식
- 날짜가 컬럼이 아니라 인덱스에 있어도 자동 변환
- `통합총자산`, `총자산`, `수익부스터평가금액`, `T100원본자산`, `전략총자산`, `Close/종가` 계열 자동 인식
- 컬럼 앞뒤 공백/BOM 제거
- 실패 시 현재 컬럼 목록을 오류 메시지에 표시

## 사용법
기존처럼 실행합니다.

```text
6. 섹터전략 백테스트
→ 기간: 2010부터 장기검증
→ 방공호 방식: T100-CAP5-R70-MIN5 현실형
→ 방공호/Turbo만 빠른 실행
```
