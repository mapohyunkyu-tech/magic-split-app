# Magic Split v45 - 기존 메인 앱 + T100-CAP5-R70-MIN5 통합

## 버전
`v45_MAIN_APP_MERGED_CAP5_R70_MIN5_20260701`

## 핵심
이 버전은 CAP5 전용 단독앱이 아니라, 기존 매직스플릿 메인 `app.py` 구조를 유지한 상태에서 `1-0) 073 / 064 / CTA 방공호+부스터 통합 백테스트` 안에 CAP5 현실형을 합친 버전입니다.

기존 메뉴/기능은 유지합니다.

- 요양원
- 운영판단기
- TOP50
- 섹터전략 백테스트
- 방공호/Turbo 빠른 실행
- 실전 운영판

## 추가된 모드
방공호 방식 선택 목록에 아래 2개가 추가됩니다.

1. `T100-CAP5-R70-MIN5 현실형`
2. `T100-CAP5-R70-MIN5 현실형 - 지수/환율 프록시`

## 처리 방식
CAP5는 손실을 당일에 잘라내는 이론형이 아닙니다.

- T100 원본 daily 수익률을 먼저 계산합니다.
- 하루 수익률이 -5% 이하이면 CAP5 신호가 발생합니다.
- 신호 당일 손실은 그대로 맞습니다.
- 다음 거래일부터 방어모드가 적용됩니다.
- 기본 방어모드: T100 70% + CASH 30%
- 최소 5거래일 유지합니다.
- 이후 T100 원본 자산곡선이 20일선 회복 또는 전고점 대비 -5% 이내 회복 시 100%로 복귀합니다.

## 설정 위치
`1-0-1) T100-CAP5-R70-MIN5 현실형 설정` expander가 추가되었습니다.

기본값:

- CAP 신호 기준 하루손실: 5%
- 방어모드 T100 비중: 70%
- 방어모드 최소 유지일: 5거래일
- 거래비용 비교: 0%, 0.1%, 0.2%
- CASH 연수익률은 기존 방공호 CASH/CD·KOFR 입력값을 사용합니다.

## 권장 실행
2010 장기검증에서는 기존 섹터전략 전체 실행보다 아래 버튼을 권장합니다.

1. `6. 섹터전략 백테스트`
2. 기간: `2010부터 장기검증`
3. 방공호 방식: `T100-CAP5-R70-MIN5 현실형` 또는 프록시형
4. `방공호/Turbo만 빠른 실행` 클릭

## 출력
빠른 실행에서는 CAP5 전용 ZIP을 다운로드합니다.

- summary
- selected_compare
- daily
- events
- mdd_episodes
- yearly_returns_mdd
- monthly_detail
- monthly_summary
- 70m_conversion

## 주의
이 버전은 기존 프로그램을 CAP5 단독앱으로 바꾸지 않습니다. CAP5는 기존 프로그램 내부의 추가 백테스트 모드입니다.
