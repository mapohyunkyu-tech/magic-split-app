# Magic Split v38 - 2010 Turbo Only Safe Run Fix

## 문제
2010년부터 백테스트를 실행하면 Streamlit에서 중간에 세션이 끊길 수 있었습니다.
원인은 T100/Turbo만 확인하려는 경우에도 앱이 먼저 섹터 개별주 백테스트 전체를 실행한 뒤 방공호/Turbo 계산을 붙였기 때문입니다.

## 수정
- `2010부터 장기검증` 기간 프리셋 추가/고정
- 최대 검증 거래일 상한 5000 유지
- 2010/2020/2021 장기검증 시작일 강제 고정
- `방공호/Turbo만 빠른 실행` 버튼 추가
- 이 버튼은 섹터 개별주 백테스트를 생략하고, ETF 방공호/Turbo 전략만 바로 계산합니다.
- 2010 장기검증 거래일은 KODEX200 네이버 장기차트를 우선 사용하고, 실패하면 평일 캘린더로 fallback합니다.

## 사용법
1. 메뉴 `6. 섹터전략 백테스트`
2. 백테스트 기간: `2010부터 장기검증`
3. 방공호 방식: `T10/T100 Turbo 공격ETF 100% NO CTA`
4. `방공호/Turbo만 빠른 실행` 클릭
5. 결과 CSV 다운로드

## 주의
`섹터전략 백테스트 실행`은 여전히 개별주 섹터전략 전체를 계산합니다. 2010 장기검증에서 끊김이 나면 Turbo 확인용으로는 쓰지 말고 `방공호/Turbo만 빠른 실행`을 사용하세요.
