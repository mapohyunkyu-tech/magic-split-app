# Magic Split v52 - 강환국 MDD5 MA `모드` 컬럼 핫픽스

## 수정 이유
`강환국식 MDD5형 MA 백테스트 실패: '모드'` 오류는 월말 신호표를 백테스트 엔진으로 넘길 때 `모드` 컬럼이 누락되어 발생했습니다.

## 수정 내용
- 강환국 MDD5 신호 생성부 `_kang_v51_generate_signals()`에 `모드` 컬럼을 항상 생성하도록 수정
- 기존 매직스플릿 메인 앱 구조 유지
- BAA/VAA, T100, CAP5, 강환국 MDD5 메뉴 유지
- 문법 검사 통과

## 실행
```bash
streamlit run app.py
```

## 확인 경로
6. 섹터전략 백테스트 → 1-0-3) 강환국식 MDD5 이동평균 자산배분 백테스트 → 강환국 MDD5 MA 빠른 백테스트 실행
